# family-kb-arduino

This project uses the ATmega32U4, specifically an Arduino Micro wired to
a male DB15 port that the HVC-007 uses as a connector.

## Pin Map
| DB15   | Arduino   |
| :----: | :-------: |
| 1      | GND       |
| 4      | 11        |
| 5      | 10        |
| 6      | 9         |
| 7      | 8         |
| 10     | 4         |
| 11     | 12        |
| 12     | 2         |
| 13     | 7         |
| 15     | 5V        |
